# 🛍️ E-Commerce Flutter App

A modern and scalable e-commerce mobile application built with **Flutter**, following clean architecture principles and best practices.

---

## ✨ Features

* 🌗 **Dark & Light Mode** support
* 🔐 Secure authentication with token storage
* 🛒 Full shopping cart functionality (add, remove, manage items)
* 🏷️ Product listing with category filtering
* 📦 Product details screen with:

  * Hero animations
  * Ratings & reviews
* 📱 Fully responsive UI using `flutter_screenutil`
* ⚡ Smooth user experience with:

  * Shimmer loading effects
  * Staggered animations
  * Lottie animations
* 🔔 Animated snack bars for feedback
* 🧠 State management using **Flutter Bloc (Cubit)**
* 🧩 Dependency Injection using **Get It**
* 🧭 Navigation using **Go Router**

---

## 🛠 Tech Stack

* **Flutter** & **Dart**
* **Flutter Bloc** (State Management)
* **Dio** + **Pretty Dio Logger** (Networking)
* **Go Router** (Navigation & Deep Linking)
* **flutter_screenutil** (Responsive Design)
* **Get It** (Dependency Injection)
* **DartZ** (Functional Programming & Error Handling)
* **flutter_secure_storage**
* **cached_network_image**
* **Shimmer**, **Lottie**, **Animated Snack Bar**

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/ahmedshawky7/ecommerce_app.git
cd ecommerce-flutter-app
```

### 2. Install dependencies

```bash
flutter pub get
```

### 3. Run the app

```bash
flutter run
```
